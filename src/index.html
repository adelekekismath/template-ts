About the Interactive Clock Application

ü§∑üèæ‚Äç‚ôÄÔ∏è What is this application?

This interactive clock application allows you to add multiple clocks to your screen, each with a different timezone. You can choose between an analog or digital clock, move and organize them as you wish, and even manually adjust the time if needed.

üí° How to use it?

  • To add a new clock, click the "Add a new clock" button.
  • To remove a clock, use the "x" button on the respective clock.
  • To personalize the time on analog clock , click on "Edit Time" button, then click on the minute's hand to adjust it.
  • You can also drag and drop clock by clicking on the clock and moving it to the desired location.

üîß Technologies Used and Why?

This application was developed in native TypeScript and CSS , without any frameworks, to optimize performance and allow full control over every aspect of the application. The use of "pure" JavaScript creates a lightweight and fast application, following a clear structure with the MVC (Model-View-Controller) pattern to separate data, display, and user interactions.
